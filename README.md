# zomb-Ai-Pazhamudir-cantai
AI based repo for Vegetable market

What's inside:

28 produce items — 12 vegetables, 12 crops/grains, 6 fruits — each with Tamil + English name and emoji icon
10 Tamil Nadu markets (Chennai/Koyambedu, Coimbatore, Madurai, Salem, Trichy, Erode, Vellore, Thanjavur, Tirupur, Dindigul) with real contact details, timing, and in-charge names
Smart form — select yield type → filters the produce dropdown and inventory grid automatically
Quantity input in Quintals / Tonnes / Kg with estimated total value shown on each card
Market result cards — ranked best price first, with a ⭐ Best Price badge, price range, timing, grade tag, and Call / View Details buttons
Contact modal — shows market in-charge name, phone, email, with a direct call button
Inventory grid — tap any item to auto-select it in the form; filterable by type tabs
Design — deep forest greens, gold accents, Playfair Display headings, Lora body text, smooth hover lifts, mobile-responsive grid, decorative header with SVG texture
